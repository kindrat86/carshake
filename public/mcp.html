CarShake — MCP Server

Valet damage proof and vehicle handover documentation app.

One endpoint. Stateless HTTP transport. No auth required. Paste it into any MCP client:

https://carshake.online/api/mcp

Install in Claude Desktop

Add to claude_desktop_config.json:

{
  "mcpServers": {
    "carshake": {
      "command": "npx",
      "args": ["mcp-remote", "https://carshake.online/api/mcp"]
    }
  }
}

Install in Cursor

Add to .cursor/mcp.json:

{
  "mcpServers": {
    "carshake": {
      "url": "https://carshake.online/api/mcp"
    }
  }
}

Inspect

npx @modelcontextprotocol/inspector https://carshake.online/api/mcp
